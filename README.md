# dtmoney
:::..Cash Flow..:::


🏁 Execute o projeto localmente:
# Clone este repositório
$ git clone https://github.com/HulkFront/dtmoney.git

# Acesse a pasta do projeto no terminal/cmd
$ cd dtmoney

# Instale as dependências
$ yarn install

# Execute a aplicação
$ yarn start
